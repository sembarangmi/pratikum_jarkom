# Laporan Praktikum Modul 9

## WEB SERVER

Pada praktikum ini dilakukan pembuatan **Web Server** sederhana menggunakan bahasa **Python**. Server berfungsi untuk menerima permintaan (**request**) dari browser, kemudian mengirimkan file HTML sebagai **response** sehingga halaman web dapat ditampilkan kepada pengguna.

### Kode HTML

![File HTML digunakan sebagai halaman web yang akan dikirimkan oleh server ketika browser melakukan permintaan ke alamat yang tersedia](../assets/image/week9%20foto2.png)

Apabila konfigurasi server dan file HTML sudah benar, maka browser akan menampilkan isi halaman **helloworld.html**, yang menandakan bahwa web server berhasil menerima request dan mengirimkan response kepada client.

## Kesimpulan

Berdasarkan praktikum yang telah dilakukan, dapat dipahami bahwa **Web Server** memiliki fungsi untuk menerima permintaan dari client, kemudian memberikan response berupa file HTML yang diminta. Dengan memanfaatkan Python, web server sederhana dapat dibuat untuk melayani permintaan dari browser melalui alamat tertentu.

Selain itu, pada latihan tambahan terlihat bahwa komunikasi antara client dan server berlangsung melalui mekanisme request dan response. Client mengirimkan permintaan ke server, kemudian server memproses permintaan tersebut dan mengirimkan halaman web kembali kepada client. Dari praktikum ini dapat disimpulkan bahwa konsep dasar web server, HTTP request, dan HTTP response telah berhasil diterapkan dengan baik.
